# Plug In C!

C bindings for developing ewwii plugins. This project allows languages like C and C++ to make plugins for ewwii.

> See [documentation](https://byson94.is-a.dev/plug-in-c.ewwii/) for more info.

## Plugc Ewwii

This is the bindings manager. We highly recommend using it to manage your C bindings.

```bash 
sh -c "$(curl -fsSL https://byson94.is-a.dev/plug-in-c.ewwii/install.sh)"
```

## Versioning

This project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

To see which version is compatible with which ewwii version, run this command:

```bash
plugc-ewwii list
```
